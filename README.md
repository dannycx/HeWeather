天气
===
采用和风天气api<br>

## 技术点
- Gson解析json数据
- okhttp请求网络数据
- MVP模式开发，将各模块分离，降低耦合度
- Google封装Room数据库的使用
- 省市县三级城市联动选择城市（底部有代码链接地址，有兴趣的可以看一下）
